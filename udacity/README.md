# Virtual Tourist Plus Food

This is an enhanced virtual tourist based on the last virtual tourist project. This app could help you plan your trip with nearby flickr photos and delicious foods from yelp.

## What's new funcationalities compared with Virtual Tourist?
* When you pin a location, you could see what's the best restaurants nearby.

## What's under hood?
* Create yelp Client API file, which contains yelp OAuth, and search restaurants request. 
* Use Core Data store Restaurant info.
* Use collection view and mapView show restaurants infomation and locations.
* Click restaurant image to see more detail about restaurant information. Such as: make a call for reservation, open webview to check more info on yelp.
